# Design State: Dashboard SaaS Moderno (Tema Claro)

## Brief Summary
- **Problem:** Tema escuro atual dificulta a leitura e análise de dados para usuários executivos.
- **Primary Persona:** Chefe (Gestão Executiva) - Busca clareza, modernidade e rapidez na análise de KPIs.
- **Success Metric:** Leitura instantânea de métricas, consistência visual absoluta e estética SaaS moderna (estilo Linear/Vercel).

## Design Principles
1. **Clareza Radical:** Os dados são a prioridade absoluta. O design deve recuar para que a informação brilhe.
2. **Hierarquia Sutil:** Uso de sombras e bordas finas em vez de cores fortes para separar seções.
3. **Acento Funcional:** Cores (Azul/Indigo) usadas apenas para indicar ação ou progresso positivo.
4. **Espaçamento Generoso:** Ar respirável entre componentes para reduzir a carga cognitiva.
